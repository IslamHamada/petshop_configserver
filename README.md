# Config Server

Central configuration service for the PetShop platform.

## Description
Serves configuration for all microservices using Spring Cloud Config.  
Reads configuration files from the `petshop_sharedconfig` repository.

## Tech Stack
- Spring Cloud Config Server  
- Java 17  
- Spring Boot 3.x
